== Description ==

FrannaWP is a lightweight WordPress Theme that uses Underscores ( _s ) framework from Autommatic (The people behind WordPress). The Theme comes packed with custom options and supports most WordPress plugins such as WooCommerce and Jetpack. Use it to build any kind of website

/**********************************************************/

== Frequently Asked Questions ==

= Does this theme support any plugins? =

FrannaWP includes support for Infinite Scroll in Jetpack.

= How many levels sub-menus the Theme support? =

FrannaWP supports up to 2 levels of sub-menus.

= Is the Theme RTL compatible?? =

Yes, FrannaWP supports RTL Direction.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
